# Matuto-Base

基于 Gin + Vue 的前后端分离管理系统框架。

## 项目介绍

Matuto-Base 是一个基于 Gin 框架和 Vue.js 的前后端分离管理系统框架,可以用于快速构建企业级中后台管理系统。

### 主要特性

- 基于 Gin 框架,提供稳定高效的后端支持
- 前端采用 Vue + Element UI,提供美观的用户界面 
- 权限管理: 基于 RBAC 的权限控制
- 代码生成: 根据数据库表自动生成增删改查代码
- 参数校验: 基于 tag 的请求参数自动校验
- 操作日志: 系统操作日志记录和查询
- 数据字典: 维护系统中的配置项和枚举值
- 定时任务: 定时任务的创建、执行和日志管理
- 服务监控: 服务器和缓存等监控
- 多数据源: 支持多数据源配置

## 技术选型

### 后端技术

- 核心框架: Gin 
- 安全框架: JWT
- 数据库: MySQL
- 缓存: Redis
- ORM: GORM
- API文档: Swagger 

### 前端技术 

- 核心框架: Vue.js
- UI框架: Element UI
- 状态管理: Vuex
- 路由管理: Vue Router
- HTTP客户端: Axios
- 富文本编辑器: Quill
